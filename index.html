Команды в GIT

  • git clone *path-to-repository* (Скопировать репозиторий на локальный диск (указать откуда копируешь))
  • git status (состояние текущего локального репозитория)
  • git add *path-to-file*(добавить новый файл под контроль ГИТа) или git add . - добавить все файлы из директории
  • git commit -m "added new file index.html" (зафиксировать изменения и дать им комментарий)
  • git push -u origin master - оправить все закоммиченные изменения в репозиторий на сервере
  • git pull - забрать свежие обновления
  • git log - показать истори изменений
  • git log --oneline - показать истори изменений в одну линию
  • git checkout ***** - посмотреть как выглядел файл до примененного комита(вместо звездочек номер коммита)
  • git checkout master - вернуться к главной ветви
  • git revert ***** - удалить изменения определенного коммита (вместо звездочек номер коммита)
  • git reset ***** - удаляет все комиты после указанного (вместо звездочек номер коммита)
  • git reset ***** --hard - удаляет все комиты, а так же все изменения после указанного (вместо звездочек номер коммита)
  • git branch ***** - создать новую ветку (вместо звездочек название ветки)
  • git checkout -b **** - создать новую ветку и сразу перейти на нее
  • git branch -a - посмотреть какие ветки есть в проекте
  • git merge ***** - объединить ветку с той которая указана вместо звездочек
  • git remote add origin git@github.com:pavel1303/git-hub-tutorial.git - подключение к удаленному репозиторию
