Команды в GIT

  • git clone path-to-repository (Скопировать репозиторий на локальный диск (указать откуда копируешь))
  • git status (состояние текущего локального репозитория)
  • git add path-to-file(добавить новый файл под контроль ГИТа)
  • git commit -am "added new file index.html" (зафиксировать изменения и дать им комментарий)
  • git push - оправить все закоммиченные изменения в репозиторий на сервере
  • git pull - забрать свежие обновления
